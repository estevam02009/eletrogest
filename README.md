# EletroGest

Sistema de gerenciamento para lojas de eletrodomésticos desenvolvido em PHP puro.

## Objetivo

Construir um sistema completo utilizando arquitetura MVC, PHP moderno e boas práticas de desenvolvimento, sem depender de frameworks.

## Tecnologias

- PHP 8.3
- Apache
- Composer
- MySQL
- Git
- GitHub

## Requisitos

- PHP 8.3+
- Composer
- Apache 2.4+
- MySQL 8+
